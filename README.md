Este projeto foi desenvolvio utilizando:

- HTML.
- CSS.
- JavaScript.

Neste projeto eu aprendi a:

- Editar CSS dinamicamente.
- Trabalhar com eventos e manipular o DOM.
- Utilizar uma biblioteca de terceiros.
